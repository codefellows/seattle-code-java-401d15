# Class 12: Spring, RESTful Routing, And Persistence

Today we'll be hooking our Spring MVC projects up to a Postgres database to save changes to data.

### Students will be able to

#### Describe and Define

- Postgres
- JPA
- Hibernate
- JDBC
- Entities
- Repositories
